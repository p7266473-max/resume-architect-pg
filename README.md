# Resume Architect PG

Resume Architect PG is a premium, client-side, interactive resume engineering tool optimized for generating ATS-compatible, professionally stylized resumes specifically designed for **MBA Candidates** and **Undergraduate Students (UG)** of all majors.

## Features

- **Double-Audience Precision Layouts**: Seamless switching between MBA profiles (with specializations like Finance, Marketing, HR, Operations, Systems) and UG branches (Engineering, Science, Arts, Commerce, BBA).
- **Tailored Suggestion Engine**: Pre-loaded, industry-vetted career summaries and impact-driven achievement bullets to align with target domain expertise (e.g. Investment Banking, Supply Chain, Software Engineering).
- **ATS-Optimized Formatting**: Clear structural hierarchies and keyword alignment.
- **Multiple Design Templates**: Real-time styling updates including Modern Executive, Slate Minimalist, and Technical Elegant.
- **Print & Export**: Configured stylesheet to automatically scale and print a pixel-perfect, clean single-page A4 document without formatting overlaps.
- **Privacy-First & Offline Ready**: All state persistence stays inside the client's local storage database; no data is ever transmitted to external servers.

## Getting Started

1. Open `index.html` directly in any web browser.
2. Select your Category (MBA / UG) and Specialization.
3. Click "Load Sample Data" to quickly see the architecture formatted with realistic career inputs, or start filling out details from scratch.
4. Customize your contact details, education history, work achievements, and key tools.
5. Choose your design template.
6. Click **Export A4 PDF** (or press `Ctrl+P`) and save as PDF. Ensure margins are set to "None" or "Default" in browser options.
